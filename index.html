学习宣传纲领草案

2020-12-17

最高纲领

团结并扩大共产主义者的队伍，在线下形成各个地区组织有力的联合。并广泛传播马克思主义，为最伟大的事业做预备。

最低纲领

在共产主义的宣传工作上有所成就，小幅提高周边的群众基础。

宣传的重要性

鉴于当今共产主义者宣传活动分散且低效理论学习较不完善。所以我们需要团结一切可以团结的来力量扩大宣传以及进行思想传播。

目前的形势：

  1. 我们没有形成联合，工人们的政治诉求和政治观点没有任何实际影响。

  2. 现在中国的主要矛盾是劳动阶级的进步诉求与反动势力的矛盾，且矛盾的主要方面是反动势力。

  3. 不管各方目的如何，大家不约而同地重新提起了毛泽东。这有助于我们借毛泽东的旗帜宣传马克思主义。

  4. 毛泽东是我们同各群体的最大的公约数。

关于宣传工作开展的原则

  1. 从实际出发，我们必须重视并做好的工作就是根据调查研究的结果结合马克思列宁主义毛泽东思想文艺宣传。

  2. 团结一切可团结的对象，传播一切可传播的、理论思想。无论对方现在是何身份，只要对方现在还有理智，我们就可以向其宣传。

  3. 宣传的首要是历史，历史故事、历史知识、历史真相；历史能够唤起人们对事实的渴求，才能就此宣传更多的政治经济学、哲学、科学社会主义原理。

  4. 关于对当局的争论。除非已经确认对方是理性的可交流的，否则不要主动说出自己对当局的看法。争吵没有意义，我们的工作还是要落实到传播思想上面来，可以多讲列宁、毛泽东关于建设社会主义的理论。

  5. 必须注重宣传质量，我们的宣传非强制性，无法做到如“青年大学习”般快速且受众广泛，所以务必在质量上超越之。

学习工作的开展

如何更好地开展学习？

主要需要组织学习的是哪些人？是理论不成熟的新同志。虽说有一定基础的同志们不能不学，但已经不太需要别人来帮助了。想要组织新人的学习，就必须要有快速全面的初步入门，我们现在还没有这样一个教材，毛泽东选集的经济学部分几乎可以说没有，马克思恩格斯、列宁的著作对于新同志们来说又长篇大论，不便阅读。曾有布哈林和普列奥布拉任斯基同志好心编了一本这样的入门教材叫做“共产主义ABC”，可惜在今天来看，这里面也有不少难以理解的过时内容。所以对于新同志展开学习所提供的帮助应当做到：

  1. 在学习中灵活运用唯物辩证法，坚持从阶级斗争的角度出发。

  2. 阐述的问题要结合实际，让人容易理解。

  3. 论证的过程简洁而有力，结合经典著作中的名句的同时还要恰当地举例。

  4. 不公开、直接的攻击任何有争议的行为，但对一切反动的观点要进行彻底的批判。

  5. 在后续阶段拟定“学习书单”，大家共同阅读学习马克思主义经典著作。

现阶段初步（线上）

（一） 宣传之具体方式方法

  1. 注意蹭热度，社论总是带来意想不到的效果。不要仅从阶级斗争角度出发，那样只能唤起群众对资产阶级的痛恨，而非资本主义本身。

  2. 语言简短有力，论述清晰明了，注重针对理论基础薄弱者的照顾。

  3. 可以利用上面其自称马列的特性打到某些目的，如视频审核不通过时以“宣传指导思想”要挟。

（二） 学习之具体方式方法

  1. 可以在群聊内部设资料管理员，组织大家通过视频、语音会议对著作展开学习讨论。

  2. 可以选取时政新闻，用马克思列宁主义的角度开展分析。

  3. 可以在一般聊天中选取争议问题，通过语音、视频的辩论解决。

  4. 可以组织通过看教学视频或技术人才专门演示等方法学习如视频剪辑，调查研究等宣传鼓动技术技能。

将来怎样解决效率问题（线下）

宣传、联合行动中出现的的效率不足是现如今共产主义者活动的一大障碍。导致这个问题的原因通常有如：

  1. 私有软件上的线上交流常常受监管

  2. 线上组织各成员所在的地区跨度太大

  3. 真正有宣传相关技术能力的人少之又少，多数还正遭受资本的奴役不好脱身来进行宣传，导致宣传内容的质量良莠不齐

  4. 对加入人员的意识形态审核较为麻烦

解决以上这些问题的办法有：

  1. 自愿适当利用支持端到端加密的开源通讯协议进行隐私安全有保障的交流：如XMPP协议等

  2. 保证安全的前提下了解并与和自己同一地区的共产主义者们取得线下的联系

  3. 面对面的在地区范围内的技术、理论学习，面对面的在地区范围内的项目协作是能够大幅度提升效率的，有时两三个人的“生产力”便可敌一个五十多人的QQ群

  4. 面对面的意识形态审核可能是世界上最方便又快捷的办法，这也是线下租住的好处之一。

组织和安全问题

  1. 各地区组织扩大后应协调好各自内部的分工和纪律性
  2. 对于新加入人员，无论其意识形态同马克思主义有何出入，秉着团结的原则，同处在共产主义旗帜下，更正其错误思想
  3. 学习，宣传，不要冲塔，冲塔只会损害集体利益。该挺身而出时便要大鸣大放，形势低迷时就要潜心打好群众基础。

怎样学习宣传马克思主义？

2020-12-16

现状

从马克思主义在劳动者中的受众程度来看，首要的任务仍然是宣传工作。宣传工作在以前开展起来并不容易，但庚子年春节以来却有较大改观。无论毛泽东思想和毛泽东主义是否被更多人学习，至少毛泽东同志作为一个象征重新收到了更多人的认可；无论人们是否开始声讨资本主义本身，至少对资本的唾弃和批判正席卷知识分子群体和无产阶级中上层。

学习宣传效率低下

现阶段中国的马克思主义者多以各社交媒体平台的群聊为组织方式，学习宣传的进行比较受阻。
学习活动的受阻主要是由于成员地域跨度太大；某些消息传递不便；某些针对性文件收到私有社交软件的监管。宣传工作的受阻主要是由于社交媒体上分工合作效率低下；技术性人才稀缺；线上的宣传方式不仅受到制约，还无法真正深入到底层劳动者中。
学习中的问题，可以通过面对面的“读书会”解决。而宣传上的问题：首先，线上宣传不能放弃，针对劳动者的宣传也要开展；其次，想要优化分工合作和技术的学习，现实世界中的交流和指导会有很大帮助。

怎么办？

据此我们提出：应当各自在各地区形成各类朋友之间10人左右的“学习宣传小组”，不仅能够到劳动者中去，还能提高学习、宣传的效率。因此我们拟好了一份“学习宣传纲领草案”供大家参考使用。链接：redpioneers.github.io
